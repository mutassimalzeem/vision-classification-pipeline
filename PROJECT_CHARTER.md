# Project Charter

## Project Name

Vision Classification Pipeline from Raw Pixels to Deployment-Ready Inference API

## Project Type

Computer Vision / Deep Learning / ML Engineering Portfolio Project

## Dataset

Kaggle Digit Recognizer

## Main Goal

Build an end-to-end computer vision classification pipeline that starts from raw flattened pixel data and ends with a deployment-ready inference design.

## Why This Project Exists

MNIST-style digit recognition is often treated as a beginner deep learning task. This project uses it differently.

The purpose is not to prove that handwritten digit recognition can achieve high accuracy. That is already known. The purpose is to demonstrate the complete workflow required to build, evaluate, document, and prepare a vision model for inference.

## Target Skills

- computer vision fundamentals
- TensorFlow/Keras workflow
- CNN architecture design
- image preprocessing
- data augmentation
- model evaluation
- error analysis
- inference pipeline design
- API contract design
- portfolio case study writing

## Project Deliverables

1. Data understanding report
2. Preprocessing plan
3. Baseline model experiment
4. CNN experiment
5. Augmented CNN experiment
6. Evaluation report
7. Error analysis report
8. Inference API design
9. Model card
10. Portfolio case study

## Out of Scope

This project will not focus on:

- advanced object detection
- segmentation
- transformer-based vision models
- production cloud scaling
- mobile deployment
- real-time video inference

## Success Criteria

The project will be considered successful if:

- the preprocessing pipeline is clearly documented
- at least one baseline model is trained
- at least one CNN model is trained
- model performance is evaluated beyond accuracy
- confusion matrix and misclassified samples are analyzed
- inference behavior is clearly defined
- the repo can be understood by a recruiter, client, or technical reviewer
