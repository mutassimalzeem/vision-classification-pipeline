# Success Criteria

## Technical Success Criteria

The project should satisfy the following:

- Raw CSV pixel data is successfully converted into image tensors.
- Pixel values are normalized.
- A train-validation split is created.
- A dense neural network baseline is trained.
- A CNN model is trained and evaluated.
- At least one augmentation or regularization experiment is performed.
- The final model is evaluated using more than accuracy.
- Misclassified samples are reviewed.
- The final model can be saved and loaded for inference.
- API request and response behavior is documented.

## Portfolio Success Criteria

The repo should clearly communicate:

- why the project was built
- what problem it solves
- what the system pipeline looks like
- what experiments were performed
- what improved the model
- where the model fails
- how it could be deployed
- what was learned

## Reviewer Success Criteria

A recruiter, client, or technical reviewer should be able to understand:

1. the problem
2. the data
3. the modeling approach
4. the evaluation method
5. the inference design
6. the project limitations
7. the next improvement steps
