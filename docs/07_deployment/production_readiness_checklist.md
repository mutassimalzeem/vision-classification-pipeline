# Production Readiness Checklist

## Data and Preprocessing

- [ ] Training preprocessing is documented.
- [ ] Inference preprocessing matches training preprocessing.
- [ ] Pixel normalization is consistent.
- [ ] Input shape is validated.
- [ ] Invalid inputs are handled.

## Model

- [ ] Final model is saved.
- [ ] Model version is recorded.
- [ ] Model card is written.
- [ ] Evaluation metrics are documented.
- [ ] Failure cases are described.

## API

- [ ] `/predict` endpoint exists.
- [ ] Request format is documented.
- [ ] Response format is documented.
- [ ] Error responses are documented.
- [ ] Confidence score is returned.
- [ ] Low-confidence behavior is defined.

## Documentation

- [ ] README explains project clearly.
- [ ] Dataset card exists.
- [ ] Experiment logs exist.
- [ ] Evaluation report exists.
- [ ] Case study exists.
- [ ] Limitations are stated honestly.

## Portfolio

- [ ] Project title is strong.
- [ ] Repo structure is clean.
- [ ] Visuals are included.
- [ ] Final results are summarized.
- [ ] LinkedIn post is prepared.
