# Portfolio Summary

## Project Title

Vision Classification Pipeline from Raw Pixels to Deployment-Ready Inference API

## One-Line Summary

Built an end-to-end computer vision classification pipeline using TensorFlow/Keras, including preprocessing, CNN training, augmentation, evaluation, error analysis, and inference API design.

## Problem

Classify handwritten digits from grayscale image data represented as raw pixel values.

## Technical Focus

- Computer vision fundamentals
- CNN modeling
- TensorFlow/Keras
- Image preprocessing
- Model evaluation
- Error analysis
- FastAPI inference design

## Why This Project Matters

The dataset is simple, but the project demonstrates the complete workflow behind a deployable vision classification system.

## Skills Demonstrated

- image tensor preparation
- baseline modeling
- CNN architecture design
- experiment comparison
- model debugging
- evaluation beyond accuracy
- inference API planning
- documentation and storytelling
