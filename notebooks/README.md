| Notebook | Purpose |
|---|---|
| `01_data_understanding.ipynb` | Load dataset, inspect shape, visualize digits |

| `03_cnn_baseline.ipynb` | Train first CNN model |

| `05_error_analysis.ipynb` | Confusion matrix and misclassified samples |
| `06_final_submission.ipynb` | Prepare Kaggle submission |

## Notebook Rules

Each notebook should include:

- objective
- imports
- data loading
- key observations
- plots
- experiment result
- short conclusion
- next step
